# WELOCME TO OMNIEVENTS REACT-NATIVE MOBILE APP

- This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## DISCLAIMER:

- THis mobile is created for a demo purpose only !
- The backend, which servers this app, is live on Render !
- And because I'm using their free services, there is some delay [cold start] on the first login ! from a few seconds up to a minute.

## INSTALLATION

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## USER STORY:

- SignUP: the signs up from the signup form, and gets a role "user" by default.
- Login: the user logs in using the sign in from.
- Only users (role), can log in to the App.
- The user can check the events list.
- The user can search a specific event from the search bar.
- The user can subscribe to event, whith clicking the button "Subscribe"
- If the user is already subscribed, The subscribe butoon will be disabled !
- Once the user is done, he can logout from the logout button on the drawer menu.
